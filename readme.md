# PixelWorldEngineSample

PixelWorldEngine的样例。

## 使用注意

- 请使用UTF-8编码

## 配置需求

- 目前支持的系统：Windows 7, Windows 8, Windows 10
- 最低内存需求：2GB
- 最低显卡需求：支持Direct 10.1
- 环境需求：Visual Studio 2017(Windows版本), Python3

## 如何配置

首先打开`gitbash`，移动到想要放置的路径，然后键入以下指令

```
git clone https://github.com/LinkClinton/PixelWorldEngineSample.git
```

由于考虑到配置方便问题，因此也将使用的第三方库上传了，因此文件可能有点大。

拷贝后，打开项目，将引擎添加进项目。之后为运行的样例添加引用以及包含路径，即引擎的头文件路径。

最后编译运行

## 第三方库

- [SDL2](http://www.libsdl.org)
- [FreeType](https://www.freetype.org)
- [glm](https://glm.g-truc.net)
